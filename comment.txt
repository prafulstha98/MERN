GIT and GITHUB commands 
//to update new added changes into a repo/repository
git add . 

//to commit the added changes to that particular repo/repository
git commit -m "updated"

//to push the changes to a particular branch
git push origin <branch-name>

//for example: the name of the branch is "main"
git push origin main 

//to create new branch on github 
git checkout -b <branch-name>
git switch -c <branch-name>

//for example: the name of the branch you want to create is "css"
git checkout -b css

//for example: the name of branch you want to switch to is: "css" 
git switch css

//lists out all the available branch you have 
git branch 

//both the commands are same and they both have same functions
git checkout -b aloo
git switch -c aloo 

//to delete a branch
git branch -d <branch-name>

//for example: suppose you have a branch named "alookhau"
git branch -d alookhau
